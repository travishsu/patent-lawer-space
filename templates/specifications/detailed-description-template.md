## WRITING TIPS

### Do's
- Use present tense
- Be consistent with terminology
- Reference figures and reference numbers
- Provide enough detail for enablement
- Include multiple embodiments
- Describe variations and alternatives
- Use "may," "can," "might" for optional features
- Use "configured to," "adapted to," "operative to" for functional language

### Don'ts
- Avoid marketing language or superlatives
- Don't limit yourself to one embodiment
- Don't use "must" or "required" unless necessary
- Avoid ambiguous antecedents
- Don't introduce new matter later
- Avoid inconsistent terminology

### Reference Number Guidelines
- Assign reference numbers systematically
- Use consistent numbering across figures
- Include reference numbers in description
- Maintain a reference number list
